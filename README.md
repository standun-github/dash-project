# dash-project

A game theory solver for two-player games.

## Description

A web application for building and solving a matrix game based on game theory approaches.


### Dependencies

* Python 3.9
* Dash 2.6.1
* Pymongo 4.4
* Dnspython 2.2.1
* Nashpy 0.0.35
* Numpy 1.23.2
* Scipy 1.9.1

## Run

* python3.9 ./main.py
